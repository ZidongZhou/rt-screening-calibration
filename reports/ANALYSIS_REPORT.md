# Additional Analysis Report

The package includes primary pipeline outputs and additional manuscript outputs. The additional analyses cover label-aware calibration, standard calibration controls, mask-aware CART/logistic comparators, high-risk burden metrics, residual diagnostics, calibration-delta bootstrap checks, and stepwise future-information perturbation auditing.

Quick checks:

```bash
python -m compileall -q src scripts tests
python -m pytest tests -q
bash run_smoke.sh
python scripts/check_expected_outputs.py
```

Complete manuscript reproduction requires both the primary and additional analysis scripts:

```bash
FORCE=1 bash run_all.sh
FORCE_ADDITIONAL=1 bash run_additional_analyses.sh
```

For checkpointed Windows or cross-platform reproduction, use
`scripts/run_reproduction_stage.py` (or `run_stage.ps1`) in the order primary,
robustness, additional, selective, publication, and checks.


Additional compact check: table14_calibration_grid_boundary_sensitivity_19.csv reports expanded sigma/temperature grid sensitivity, including PSS upper-bound behavior and test-set probability-summary changes.

## Selective auditing analyses

The package includes `scripts/selective_auditing_analyses.py`, which creates
participant-by-scale decision audit frames at the 19-item budget and evaluates
error detection, overconfident-error detection, process-quality strata,
leave-one-scale-out transfer, selective full-form escalation, bootstrap
uncertainty, threshold sensitivity, and per-scale escalation diagnostics.
Final numbering and labels are generated by
`scripts/12_export_frontiers_submission_outputs.py`. The results support a
conservative interpretation: RT features are useful as process-audit signals,
especially for overconfident-error and process-quality review, but they do not
establish RT-specific calibration superiority, scale-invariant transfer, or
stable superiority for elevated-score false-negative capture.
