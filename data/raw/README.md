# Raw data

Raw CSV files are not stored in this repository. To rerun the full pipeline, download Zenodo record DOI `10.5281/zenodo.10423537` (version v2, CC BY 4.0) and place these files here:

- `demographic.csv`
- `phq9.csv`
- `gad7.csv`
- `pss.csv`
- `isi.csv`

Expected MD5 checksums:

- `demographic.csv`: `7a567dc5a9cf1c6a7ea2b70f166dbb57`
- `gad7.csv`: `7025495755569edd7dd5561a706178fb`
- `isi.csv`: `b9976f71c4db473dd31e4b12b40cf592`
- `phq9.csv`: `585000bbbec153efde405b3442eda479`
- `pss.csv`: `91d7c288095213d50f62adcfad15117e`

The aggregate tables and figures needed for manuscript checks are already under `results/frontiers_submission`.
