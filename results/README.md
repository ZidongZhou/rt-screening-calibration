# Results directory map

Use `results/frontiers_submission/` for the display items cited in the manuscript. It contains Tables 1-10, Supplementary Tables S1-S11, Figures 1-4, and Supplementary Figures S1-S4 with the final Frontiers numbering.

Files in `results/tables/`, `results/figures/`, and `results/supplementary_figures/` are internal or extended pipeline outputs. Some retain legacy numbering from the full reproducibility pipeline and are not manuscript display-item numbering.
